What is DOM and it's importance

DOM stands for Document Object Model, which is a programming interface for web documents. It represents the HTML or XML document as a tree structure, where each node in the tree corresponds to a part of the document, such as an element, attribute, or text node.

The purpose of the DOM is to provide a way for programs to access and manipulate the content and structure of a web page. It allows web developers to dynamically update the content of a page without reloading the entire page.

How to select element from DOM?

To select HTML elements using the DOM, there are several methods available. Here are some of the most commonly used DOM methods for selecting elements:
1. getElementsByTagName()
2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()

What is Event Bubble?

Event bubbling is a mechanism in which an event that is triggered on an element is propagated up the DOM tree, from the target element to its parent elements, all the way up to the document object.

When an event is triggered on an element, it first fires on the element itself, then bubbles up to its parent elements in the DOM tree until it reaches the document object.

What is Callback Function and how it is used?

In programming, a callback function is a function that is passed as an argument to another function and is executed by that function at a certain point in its execution. The purpose of a callback function is to allow the caller function to execute some code or perform some action after the execution of the called function is complete.
